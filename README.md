# Expense Tracker

## Description
A full-stack expense tracker application built using Node.js, Express, and SQLite.
The system allows users to manage expenses through Create, Read, Update, and Delete (CRUD) operations using a Node.js and Express backend, with a simple client interface.
## Features
- REST API with CRUD operations
- Client-side UI
- API documentation using APIDOC
- Functional testing with TestCafe

## How to Run
Install Dependencies: npm install
To start the application, run: npm start
Server will run on: http://localhost:3000
Run Client Application: public/index.html
Run TestCafe tests: npx testcafe chromium tests/
